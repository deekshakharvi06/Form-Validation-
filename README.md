# Form-Validation-
Registration Form Validation using HTML, CSS, Bootstrap and JavaScript

# Tech Stack

HTML5 – Structure

CSS3 / Bootstrap 5 – Styling and responsive layout

JavaScript – Client-side validation logic

# Field Constraints

Full Name : 
    Only letters and spaces allowed.
    No numbers or special character.
    Max 50 characters.

Email :  
    Must follow standard email format (e.g., name@gmail.com ).

Phone Number : 
    10 digits only.
    Starts with 7, 8, or 9.
    Fixed "+91" code before input.

Password : 
    First letter uppercase
    Must include lowercase, number, and special character.
    Minimum 8 characters.
    Shows live checklist while typing.

Confirm Password : 
    Must match the entered password.
    Password checklist hides on focus.
